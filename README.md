# IRLite

A Fabric client mod (MC 1.20.x) — an addon for [BBS](https://github.com/mchorse/bbs-mod) that adds dynamic point and spotlight light sources with per-light volumetrics and real-time shadows, consumed by Iris shaderpacks. Ships a patcher that injects the required GLSL into a target shaderpack.
